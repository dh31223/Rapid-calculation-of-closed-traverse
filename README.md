# 闭合导线计算

## 项目来源：

    在计算闭合导线时往往会有大量重复且繁琐的运算，为了提升运算效率，制作了该小程序

## 主要功能：

    输入实际测量的各种参数，然后会生成相应的闭合导线数据表

    输入计算的角度，可以计算出改正数，提升测量的准确性

    可以对度分秒形式的角度进行简单运算，提升实际工作的计算效率

## 项目心得：

    作者在学习CPP面向对象和数据结构时刚好遇到计算闭合导线这种繁琐的问题，所以想结合所学知识制作一个可以提升工作效率的程序；

    为了在程序中运用所学知识以达到熟练使用数据结构的目的，制作时并没有太多的考虑程序的简洁性；

    作者的初心是想在解决闭合导线计算问题的同时提升自己对使用面向对象和数据结构的熟练度；

## 主要语言：

    C++
